Definiciones clave

Basado en su documentación interna y fuentes especializadas, resumimos los conceptos esenciales para aplicaciones empresariales.

Espacio muestral (Ω): conjunto de todos los resultados posibles de un experimento aleatorio (ej.: lanzar un dado Ω = {1,2,3,4,5,6}).

Evento: subconjunto de Ω. Ej.: A = {4,5,6} es el evento "turno prioritario" si el dado sale 4–6.

Probabilidad: medida entre 0 y 1 asignada a eventos; P(Ω)=1 y para A⊂Ω, P(Aᶜ)=1−P(A).

Ejemplos resueltos en logística (detallado)

1) Demanda diaria de gasolina

Espacio muestral: Ω = {0,1,2,…,1000} galones diarios (discreto). Evento de interés: "demanda superior a 800 galones" A = {801,...,1000}.

Si no tenemos otra información y asumimos distribución uniforme (hipótesis conservadora para estimaciones iniciales), la probabilidad teórica sería:

P(A) = |A| / |Ω| = 200 / 1001 ≈ 0.1998 (≈19.98%)

Interpretación logística: Con P(A)≈20%, planificar reabastecimiento temprano en 1 de cada 5 días ayuda a mantener servicio; ajustar con datos históricos para obtener distribución real (no uniforme).

2) Inspección de lotes

Se inspeccionan 5 lotes; cada lote puede ser {defectuoso, aceptable}. Espacio muestral: |Ω| = 2^5 = 32 combinaciones posibles.

Evento: "al menos 2 defectuosos". Si p = probabilidad de defecto por lote (estimada con histórico), la probabilidad se calcula por distribución binomial:

P(X ≥ 2) = 1 − [P(0) + P(1)] = 1 − [ (1−p)^5 + 5·p·(1−p)^4 ]

Ejemplo numérico: si p = 0.1 (10% defecto),

P(X ≥ 2) = 1 − [0.9^5 + 5·0.1·0.9^4] ≈ 0.08146 (≈8.15%)

Aplicación: si P(X ≥ 2) supera un umbral aceptable (ej. 5%), activar contención y revisar proveedores.

3) Lanzamiento de dado para turnos

Espacio muestral: Ω = {1,2,3,4,5,6}. Evento "turno prioritario": A = {4,5,6}. P(A) = 3/6 = 0.5.

Interpretación: si asignas prioridad con ese criterio, la mitad de los días habrá turno prioritario; útil para balancear carga operativa.
